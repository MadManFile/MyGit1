## 🎯 هدف این مخزن

یاد گرفتن **GitHub Flow** و **خودکارسازی با GitHub Actions** از طریق تمرین عملی.

## 📋 چه چیزهایی یاد می‌گیرم؟

| موضوع | وضعیت | توضیحات |
|:------|:-----:|:---------|
| ایجاد شاخه (Branch) | ✅ | یاد گرفتم چطور شاخه جدید بسازم |
| Pull Request | ✅ | تمرین درخواست کشیدن و مرور کد |
| Merge | ✅ | ادغام تغییرات در شاخه اصلی |
| GitHub Actions | ⏳ | در حال یادگیری خودکارسازی |
| Branch Protection | ⏳ | تنظیمات محافظت از شاخه اصلی |

## 🛠️ مراحل تمرین

1. ✅ مخزن را ساختم
2. ✅ این README را نوشتم
3. ⏳ یک شاخه جدید ایجاد می‌کنم
4. ⏳ تغییرات را Commit می‌کنم
5. ⏳ Pull Request باز می‌کنم
6. ⏳ Pull Request را Merge می‌کنم
7. ⏳ اولین GitHub Action را می‌نویسم
